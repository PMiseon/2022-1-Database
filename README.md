# DBMS, 개발언어 및 개발도구 
- DBMS : MySQL 8.0.15 8.0.15 혹은 이상 버전
- 개발언어 이상 : JAVA (Oracle JAVA SE 11 )
- 개발도구 : Eclipse

# 개발 환경 
- 사용자 인터페이스는 반드시 GUI(Graphical User Interface)를 사용한다.
- 단 GUI 구동에 필요한 라이브러리는 기본 JDK에 포함되어 있는 것으로 한정한다.
